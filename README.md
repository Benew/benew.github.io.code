# benew.github.io
My portfolio

# Contributing

## Get started
```bash
npm install -g gulp
npm install
gulp
```

## Publish

This will build and push to the right repository : [https://github.com/Benew/benew.github.io](https://github.com/Benew/benew.github.io)

```bash
make
```
